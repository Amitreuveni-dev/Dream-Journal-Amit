# Dream Journal

A full-stack Dream Journal application built with **React**, **TypeScript**, and **SCSS**, using a clean client–server architecture.  
The client provides a modern and intuitive interface for writing and browsing dream entries, while the backend manages validation, data handling, and API logic.  
This project demonstrates end-to-end development skills, strong typing, and scalable application structure.

---

## Features
- Add, view, and manage dream entries  
- React-based UI with TypeScript for type-safe components  
- SCSS styling with modular and maintainable structure  
- Backend API built with TypeScript  
- Clear separation between client and server layers  

---

## Tech Stack
**Client:** React, TypeScript, SCSS  
**Server:** TypeScript (Node)

## Getting Started

Clone the repository:

git clone https://github.com/Amitreuveni-dev/Dream-Journal-Amit.git
cd Dream-Journal-Amit

Install dependencies (client + server):

cd client
npm install

cd ../server
npm install

Run the client and server:
npm start
